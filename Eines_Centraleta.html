Introducció al Projecte

La biodiversitat vegetal del campus de la UAB representa un patrimoni natural i educatiu de gran valor, amb funció de corredor biològic entre espais naturals propers. Aquest projecte desenvolupa una eina digital moderna per catalogar i difondre la flora del campus utilitzant tecnologies obertes.

Objectius del Projecte

🎯 Objectiu General

Dissenyar i implementar un ecosistema digital basat en tecnologies obertes que permeti catalogar, gestionar i divulgar la biodiversitat vegetal del campus de la Universitat Autònoma de Barcelona de manera interactiva i accessible.

Objectiu Descripció Estat
OE-1 Plugin WordPress amb galeria filtrable i mapa interactiu Leaflet ✅ Assolit 100%
OE-2 Aplicació gestió catàleg amb File System Access API ✅ Completat
OE-3 Base de dades amb mínim 150 espècies del campus ✅ >300 espècies
OE-4 Avaluació usabilitat SUS ≥80 amb 50+ usuaris ✅ SUS >94 (90 usuaris)
OE-5 Aplicació d'edició integrada amb funcionalitats CRUD ✅ Revisat i provat
OE-6 Memòria TFG i defensa oral dins del termini oficial ✅ Completat

Desenvolupament del Sistema

🔌 Plugin WordPress

Shortcodes Implementats:

  • [galeria_botanica] - Galeria visual filtrable
  • [mapa_botanica] - Mapa interactiu Leaflet

Funcionalitats Avançades:

  • Filtres dinàmics per família, hàbitat i colors
  • Cerca instantània amb autocompletat
  • Lazy-loading d'imatges optimitzat
  • Deep-linking per compartir espècies
  • Disseny responsive i accessible
  • Lightbox amb navegació per teclat

⚙️ Gestor de Catàleg

Gestió Local Avançada:

  • Edició local de plantes.json
  • Validació en temps real de camps
  • Gestió d'imatges amb drag-and-drop
  • Mapa Leaflet integrat per coordenades
  • File System Access API opcional
  • Treball offline complet
  • Previsualització instantània

📊 Arquitectura del Model de Dades

🏷️ Identificació Taxonòmica

  • Identificador únic (slug)
  • Nom comú català
  • Nom científic binomial
  • Família botànica
  • Tipus vital (arbre, arbust, etc.)

🌱 Característiques Biològiques

  • Hàbitat preferent
  • Període de floració
  • Tipus de fullatge
  • Alçada màxima
  • Colors representatius

📍 Dades Espacials

  • Coordenades GPS múltiples
  • Visualització en mapa
  • Clustering automàtic
  • Pop-ups informatius
  • Navegació entre punts

💡 Impacte i Sostenibilitat

Aquest projecte demostra que amb una combinació intel·ligent de tecnologies web obertes i una metodologia centrada en l'usuari es pot crear una eina robusta i atractiva per la divulgació del patrimoni natural universitari.
